# LLM logos

Drop logo images here to use in the AI-Agnostic section. The app looks for **PNG** files first (then SVG if PNG is missing) by slug:

- `openai.png` — OpenAI / GPT-4o
- `claude.png` — Anthropic Claude
- `gemini.png` — Google Gemini
- `llama.png` — Meta Llama 3.2
- `deepseek.png` — DeepSeek
- `local.png` — Local / On-Prem

Use PNG or SVG; name files with the `.png` extension (e.g. `openai.png`). Roughly square, transparent background works best. If a file is missing, the section falls back to the monospace abbreviation.
