# Картинки для карусели Use Cases

Положите сюда изображения для карточек карусели. Имена файлов должны совпадать с указанными ниже — тогда они подставятся автоматически.

| Файл | Карточка |
|------|----------|
| `enterprise.png` | Employee Experience (Enterprise) |
| `edtech.png` | Learning & Training (EdTech) |
| `cx-support.png` | Customer Experience (CX / Support) |
| `healthcare.png` | Mental Health & Care (Healthcare) |
| `sales-marketing.png` | Conversion & GTM (Sales & Marketing) |
| `robotics.png` | Companion Robotics (Robotics) |
| `automotive.png` | In-Car Voice Assistants (Automotive) |

Формат: PNG или JPG. Рекомендуемый размер: около 340×255 px (пропорция 4:3).

Если файла нет — в карточке показывается плейсхолдер с иконкой.
