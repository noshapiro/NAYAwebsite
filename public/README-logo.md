# Логотип сайта

Положите файл логотипа **NEARU** в эту папку:

- **Папка:** `nearu-site/public/`
- **Имя файла:** `logo.png`

Итоговый путь: **`nearu-site/public/logo.png`**

Логотип используется в шапке (Navbar) и в подвале (Footer).  
Если файла нет, показывается запасной вариант (синяя точка + текст NEARU).
