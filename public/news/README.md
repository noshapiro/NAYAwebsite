# News images

To show only the **image** from a LinkedIn post (not the full embed), save the image file here and set the `image` field in `src/data/news.ts`.

**For the "AI memory types" post:**  
Save the infographic from the post as `ai-memory-types.png`:
1. Open the post on LinkedIn.
2. Right-click the diagram image → "Save image as…" (or equivalent).
3. Put the file in this folder: `public/news/ai-memory-types.png`.

The card will then display only that image; "Read on LinkedIn" still links to the full post.
